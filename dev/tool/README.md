# FDC CookieLaw Tool (0.11.0-alpha)
Plugin jQuery per la gestione delle pagine necessarie agli adempimenti della CookieLaw

* Informativa breve sull'uso dei cookie (banner)
* Informativa estesa sull'uso dei cookie
* Privacy Policy
